Saya memiliki website utama yang sudah memiliki sidebar dan dashboard existing.

Saya ingin MENAMBAHKAN fitur baru berupa:

"Shopee Admin Panel"

Konsepnya:
- Di sidebar utama website tambahkan menu baru:
  - Shopee Admin
- Ketika diklik:
  - menuju halaman login admin Shopee
- Setelah login berhasil:
  - masuk ke dashboard khusus Shopee
- Dashboard Shopee terpisah dari dashboard utama website
- Dashboard Shopee memiliki sidebar sendiri khusus Shopee API

PENTING:
- Jangan merusak layout website utama
- Jangan overwrite CSS existing
- Jangan membuat ulang dashboard utama
- Hanya tambahkan module baru
- Gunakan style/theme existing website
- Gunakan component/sidebar existing jika memungkinkan
